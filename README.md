# Simulacion-de-Sistemas-Proyecto-Plaza-Vea
Modelo de simulación del sistema de colas en las cajas de atención del supermercado Plaza Vea. El proyecto utiliza teoría de colas y simulación de eventos discretos para analizar el rendimiento del sistema, determinar el número óptimo de cajas operativas para reducir los tiempos de espera de los clientes y optimizar los recursos.
